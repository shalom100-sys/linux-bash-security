# linux-bash-security

This repository documents my learning path in Linux and Bash scripting
with a focus on security fundamentals and system automation.

## Goals
- Build strong Linux command-line foundations
- Learn Bash scripting step by step
- Understand authentication logic and access control
- Practice automation and system cleanup
- Prepare groundwork for security research and exploit development

## Repository Structure
The repository is organized by learning stages:

- `01-basics` – Linux commands, files, permissions, and basic scripts
- `02-authentication` – Login logic, conditionals, and failure handling
- `03-loops-and-automation` – Loops, attempts counters, automation
- `04-system-enum` – System enumeration and information gathering
- `05-cleanup-and-hardening` – Cleanup scripts and performance basics
- `notes` – General concepts and learning notes

## Environment
- OS: Kali Linux
- Shell: Bash
- Focus: Security-oriented scripting

## Status
This repository is actively updated as I progress and learn new concepts.
