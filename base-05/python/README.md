# Python Login System

## Overview

This project implements a basic login system in Python.
It demonstrates fundamental programming concepts while introducing core ideas related to authentication and security.

---

## Features

* User input for username and password
* Basic credential validation
* Clear success / failure output
* Modular structure using a function

---

## Technologies

* Python 3

---

## Usage

Run the script:

```bash
python3 login.py
```

---

## Project Structure

```
python/
└── login.py
```

---

## Learning Objectives

* Understand input/output operations in Python
* Use conditional logic (`if`, `else`)
* Work with functions
* Build a simple authentication flow

---

## Security Perspective

This implementation is intentionally simple and not secure.
It is designed for learning purposes.

Key limitations:

* Credentials are hardcoded
* No encryption or hashing
* No protection against brute force attacks
